# Yaml
YAML: YAML Ain't Markup Language™ 是一个数据序列化语言（data serialization language），适合人类读写，以及跨编程语言。

是 [json](json.md) 的严格超集（strict superset），意味着一个 json 文件可以被解读为 yaml 文件（存疑？）
